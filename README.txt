Voici le repository GitHub de mon projet, un moteur de création de jeux vidéos 2D sur le Web.
L'installation est simple, vous devez tout simplement avoir un logiciel tel que Wampserver pour pouvoir faire fonctionner
le serveur.
Ensuite vous placez le dossier game-engine dans le dossier de travail de votre serveur local (dossier 'www' avec Wamp).
Enfin, vous lancez le fichier "client/index.html" et vous pouvez tester l'éditeur.

Version actuelle : Développement

Actions possibles : 
    -Dessiner une map
    -Sauvegarder une map
    -Rouvrir une map sauvegardée
    -ATTENTION : Pour pouvoir rouvrir une map à tous les coups il faut désactiver le cache de votre navigateur
    -Modifier les propriétés
